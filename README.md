# Quickstart

## Example

#### Code

```c
// code.c
#include <stdio.h>
int main()
{
  printf("Hello World!\n");
  return 0;
}
```
#### Makefile

```Makefile
# Makefile
all: hello
hello: code.o
  gcc code.o -o hello
code.o:
  gcc -c -Wall -Wextra -Werror -std=c99 -pedantic -g code.c
clean:
  rm -rf *o hello
```

#### Compile

    $ make
    
#### Run

    $ ./hello
    Hello World!
